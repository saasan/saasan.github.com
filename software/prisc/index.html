プリスク : DMM版(Windows PC版)プリンセスコネクト！Re:Diveをフルスクリーン化

2019/09/08

プリスク

DMM版(PC版)プリコネRをフルスクリーン化

ダウンロード

prisc.exe

使い方

  1. DMM版プリコネRを起動
  2. プリコネRのウィンドウが表示されている状態で prisc.exe を実行
  3. ユーザーアカウント制御の確認画面が出たら [はい] を選択
  4. プリコネRのウィンドウがフルスクリーン化される
  5. プリコネRを終了する場合は Alt + F4 キーを押す

注意点

ディスプレイの解像度が 1920x1080 または 1280x720 の場合のみフルスクリーン化できることを確認しています。 その他の解像度ではフルスクリーン化できない可能性があります。
